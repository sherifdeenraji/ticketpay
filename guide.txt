# OAU Digital Transport Ticketing System

## Project Requirements & Functional Specification

### Background

The current transport system in Obafemi Awolowo University (OAU) operates using a prepaid ticket model. Students purchase physical transport tickets before boarding buses or tricycles, and transport operators collect these tickets from passengers. OAU recently introduced a centralized cashless transport framework with a strict "No Cash" policy, requiring passengers to obtain tickets before boarding. Tickets currently cost ₦100 per ride ("per drop").

This project aims to replace the physical ticket system with a fully digital payment platform that allows students to pay for transport rides directly from their phones.

# Ride Payment System

Instead of buying physical tickets, users will pay digitally.

---

## Driver Code Payment

Every registered driver receives:

* Unique Driver Code
* Example:

DRV001
DRV002
DRV003

Payment flow:

1. User boards vehicle.

2. Driver provides code.

3. User enters code.

4. System validates code.

5. User selects number of tickets:

   * 1 Ticket
   * 2 Tickets
   * 3 Tickets
   * Custom Quantity (Optional)

6. System calculates amount automatically.

Example:

Ticket Price = ₦100

2 Tickets = ₦200

3 Tickets = ₦300

7. User confirms payment.
8. Amount deducted from wallet.
9. Driver receives payment record instantly.

---

## QR Code Payment

Alternative payment method.

Each driver has:

* Unique QR Code

Payment flow:

1. User taps "Scan QR".
2. Camera opens.
3. Scan driver's QR Code.
4. Driver information loads.
5. User selects ticket quantity.
6. Wallet charged.
7. Payment completed.
